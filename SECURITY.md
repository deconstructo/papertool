# Security Policy

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, email **metanoia@gmail.com** with the subject line `[SECURITY] papertool`.

Include as much of the following as you can:
- Type of issue (e.g. XSS, dependency vulnerability, exposed secret)
- File paths or URLs relevant to the issue
- Steps to reproduce
- Potential impact

You should receive a response within 48 hours.
